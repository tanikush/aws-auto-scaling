# GitHub pe Upload Karne ke Steps

## 1. Apni Images Copy Karein
Apni saari images ko `images` folder mein copy kar dein:
- C:\Users\TANISHA\aws-auto-scaling-learning\images\

## 2. Images ka naam README ke according rakhein (optional):
- architecture.png
- config-steps.png
- dashboard.png
- scaling-activity.png
- cloudwatch-metrics.png

(Ya apne images ke naam ke according README.md mein update kar dein)

## 3. Git Initialize karein
```bash
cd C:\Users\TANISHA\aws-auto-scaling-learning
git init
git add .
git commit -m "Initial commit: AWS Auto Scaling documentation"
```

## 4. GitHub pe Repository banayein
- GitHub.com pe jao
- "New Repository" click karein
- Repository name: aws-auto-scaling-learning
- Public select karein
- "Create repository" click karein

## 5. Local repository ko GitHub se connect karein
```bash
git remote add origin https://github.com/YOUR-USERNAME/aws-auto-scaling-learning.git
git branch -M main
git push -u origin main
```

## 6. LinkedIn Post mein Link add karein
Repository ka link apni LinkedIn post mein update kar dein!
